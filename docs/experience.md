# Work Experience

## Full-Stack Developer — Latin American Association of Kazan (March 2025 – present, 1 year)

Maria Juliana independently built and maintains the association's web platform,
serving the Latin American student community in Kazan. She is the sole developer,
responsible for the entire cycle: gathering requirements from the association's
board, development, deployment and ongoing maintenance.

The platform is built with Next.js 15, React with TypeScript and Supabase, and is
fully trilingual (Spanish, English, Russian) using next-intl. She implemented a
multilingual AI chatbot powered by Cloudflare Workers AI (Llama 3.1 8B) with a
dynamic system prompt built from live Supabase data — it answers questions about
country representatives, university representatives, migration procedures and
events in real time. She also built a protected admin panel for managing content
directly from the database, custom animations, and deployed everything on Vercel
with automatic CI/CD. The site has been live in production with real users since 2025.
