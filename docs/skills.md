calibration, computer vision with YOLOv8, and dataset collection and labeling.

Tools and infrastructure: Docker and Docker Compose, Git and GitHub, CI/CD with Vercel,
Prometheus and Grafana monitoring, Postman for API testing, i18n with next-intl.
